什么是 Markdown

Markdown 是一种轻量标记语言，允许人们“使用易读易写的纯文本格式编写文档，然后转化为有效的HTML文档“

Markdown 仅需要几个特殊的字符（＃，＊，－…）就可以格式化字符，段落，列表等等。

Markdown 举例

字符

样式 语法 效果
粗体 ＊＊字符＊＊ 字符
斜体 ＊字符＊ 字符
删除 ～～字符～～ 字符
粗斜体 ＊＊字char符＊＊ 字char符

引用

样式 语法 效果
引用 >字符 >字符

列表

行首使用－、＊或者数字+. 来表示列表的一行，嵌套列表可以在行首加两个空格来表示，实例：
语法
1. Make my changes
2. Fix bug
3. Improve formatting
* Make the headings bigger
4. Push my commits to GitHub
5. Open a pull request
6. Describe my changes
7. Mention all the members of my team
* Ask for feedback
